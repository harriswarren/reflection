# Splats folder

Place Marble Gaussian splat files (`.spz`) here.

- **Simple-Island.spz** — used by default for the WebXR demo when `VITE_SPLAT_URL` is not set (see `src/config.ts`).
- Or set `VITE_SPLAT_URL` in `.env` to a path (e.g. `/splats/your-file.spz`) or a full URL from the Marble API.

See **docs/TROPICAL_ISLAND_WEBXR_STEPS.md** for the full flow.
